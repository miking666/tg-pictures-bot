# Инструкция по эксплуатации

## Требования

- Python 3.12+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- Google Gemini API Key (получить в [Google AI Studio](https://aistudio.google.com/apikey))

## Установка

```bash
# Клонировать репозиторий
git clone <url> && cd tg_pictures_bot

# Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
# .venv\Scripts\activate    # Windows

# Установить зависимости
pip install -r requirements.txt
```

## Конфигурация

Скопировать `.env.example` в `.env` и заполнить:

```bash
cp .env.example .env
chmod 600 .env        # ограничить доступ к файлу с секретами
```

### Обязательные переменные

| Переменная     | Описание                    |
|----------------|-----------------------------|
| `BOT_TOKEN`    | Токен Telegram-бота         |
| `GEMINI_API_KEY` | API-ключ Google Gemini    |

### Необязательные переменные

| Переменная             | По умолчанию                        | Описание                                                  |
|------------------------|-------------------------------------|-----------------------------------------------------------|
| `GEMINI_TEXT_MODEL`    | `gemini-2.5-flash`                  | Модель для обработки текста и распознавания речи           |
| `GEMINI_AUDIO_MODEL`  | `gemini-2.5-flash`                  | Модель для транскрибации голосовых сообщений               |
| `GEMINI_IMAGE_MODEL`  | `gemini-2.5-flash-image`            | Модель для генерации изображений                          |
| `DATA_DIR`             | `.data`                             | Каталог для хранения данных (бюджет, FSM-состояние)       |
| `BUDGET_FILE_PATH`     | `<DATA_DIR>/budget.json`            | Путь к файлу бюджета                                     |
| `FSM_STORAGE_PATH`     | `<DATA_DIR>/fsm_storage.json`       | Путь к файлу хранения состояний FSM                       |
| `CARD_FONT_PATH`       | `fonts/LiberationSans-Bold.ttf`     | Путь к TTF-шрифту с поддержкой кириллицы                  |
| `CARD_IMAGE_SIZE`      | `1024`                              | Размер открытки в пикселях (квадрат, минимум 512)         |
| `MAX_GREETING_LENGTH`  | `350`                               | Максимальная длина текста поздравления (символов)          |
| `MAX_BUDGET_USD`       | `3.0`                               | Месячный лимит расходов на Gemini API (в долларах)        |
| `IMAGE_GENERATION_COST`| `0.10`                              | Стоимость одной генерации изображения (в долларах)        |
| `ADMIN_CHAT_ID`        | *(пусто)*                           | Telegram chat ID администратора для уведомлений и статистики |

## Запуск

### Быстрый старт (одна строка)

Если `.env` уже заполнен:

```bash
python -m venv .venv && .venv/bin/pip install -q -r requirements.txt && .venv/bin/python bot.py
```

### Стандартный запуск

```bash
source .venv/bin/activate
python bot.py
```

Бот запустится в режиме long polling. Логи выводятся в stdout.

### Запуск через systemd (рекомендуется для сервера)

Создать файл `/etc/systemd/system/tg-pictures-bot.service`:

```ini
[Unit]
Description=Telegram Pictures Bot
After=network.target

[Service]
Type=simple
User=<ваш_пользователь>
WorkingDirectory=/path/to/tg_pictures_bot
ExecStart=/path/to/tg_pictures_bot/.venv/bin/python bot.py
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl daemon-reload
sudo systemctl enable --now tg-pictures-bot
sudo journalctl -u tg-pictures-bot -f   # просмотр логов
```

## Использование бота

### Команды

| Команда    | Описание                                      |
|------------|-----------------------------------------------|
| `/start`   | Начать работу с ботом                         |
| `/help`    | Показать справку                              |
| `/cancel`  | Сбросить текущий сценарий                     |

### Сценарий создания открытки

1. Отправить боту **текст поздравления** или **голосовое сообщение**
2. Бот обработает текст (при необходимости распознает речь и уточнит формулировку)
3. Выбрать **стиль открытки** из предложенных кнопок
4. Дождаться генерации (до минуты)
5. Получить готовую открытку в виде фото

### Доступные стили

| Стиль          | Описание                                |
|----------------|----------------------------------------|
| Цветы          | Элегантный цветочный натюрморт          |
| Портрет        | Художественный портрет с человеком      |
| Животные       | Милое животное в праздничной обстановке |
| Фэнтези        | Магическая фэнтезийная сцена            |
| Праздничный    | Праздничная сцена с подарками           |
| Пейзаж         | Живописный пейзаж                       |
| Иллюстрация   | Яркая мультяшная иллюстрация            |
| Уютный         | Тёплая уютная атмосфера                 |

## Администрирование

### Аккаунт администратора

Если указан `ADMIN_CHAT_ID`, бот будет:
- отправлять **статистику** использования (обновляется после каждой генерации)
- присылать **предупреждения** при достижении 80% и 100% бюджета
- уведомлять об **ошибках** с трейсбеком

Аккаунт администратора **не может** использовать бота для генерации открыток (блокируется фильтром `StatsAccountFilter`).

### Бюджет

Бот ведёт учёт расходов в скользящем окне **30 дней**. Данные хранятся в `budget.json` (путь задаётся через `BUDGET_FILE_PATH`).

- При достижении лимита (`MAX_BUDGET_USD`) генерация блокируется
- Если Gemini API недоступен, бот создаёт **fallback-открытку** с локальным фоном (бесплатно)
- Устаревшие события (старше 30 дней) автоматически удаляются

### Структура данных

```
.data/
  budget.json        — учёт расходов и статистика
  fsm_storage.json   — состояния диалогов пользователей
```

Оба файла создаются автоматически при первом запуске. Запись атомарная (через временный файл + `os.replace`), файлы защищены файловой блокировкой (`flock`).

## Устранение неполадок

| Проблема | Решение |
|----------|---------|
| `Missing required environment variables` | Проверить что `BOT_TOKEN` и `GEMINI_API_KEY` заданы в `.env` |
| `CARD_FONT_PATH does not exist` | Убедиться что файл шрифта существует по указанному пути |
| `Budget exceeded` | Увеличить `MAX_BUDGET_USD` или дождаться обновления 30-дневного окна |
| Бот не отвечает администратору | Это нормально — аккаунт `ADMIN_CHAT_ID` заблокирован от использования бота |
| Открытка с локальным фоном | Gemini API был недоступен; бот использовал резервный режим |
| Ошибка транскрибации | Бот делает 3 попытки; убедиться что голосовое сообщение чёткое |
